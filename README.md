## Описание архитектуры

В игре существует три вида персонажей, каждый из которых наследуется от 
абстрактного класса `Character`:

* `Player` - игрок. Создается фабричным методом, инициализируется 
сохраненными для данного игрока параметрами и набором предметов.
* `Human` - человек. Создается фабричным методом, инициализируется случайным 
именем, предметами.
* `Enemy` - враг. Создается фабричным методом с указанием типа и силы врага, 
инициализируется случайными значениями, соответствующими данному типу и силе
врага, определенными в `EnemyConstants.h`. Существует 3 вида врагов, все они - 
наследники `Enemy`:
    * `Forest demon`
    * `Hovering beast`
    * `Poisonous monster`

Диаграммы с иерархией классов находятся в `InheritanceGraphs`.
